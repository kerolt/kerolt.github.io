简记

靠右的const是顶层const，靠左的const是底层const

顶层const

即指针本身是个常量

出错原因是因为pa1是一个顶层const，即其只能指向a，无法修改其指向b。

底层const

即指针所指的的对象是个常量

pa2是一个底层const，即pa2所指对象的值不可改变，但是pa2所指向的对象可以改变

注意：
用于声明引用的const都是底层const
